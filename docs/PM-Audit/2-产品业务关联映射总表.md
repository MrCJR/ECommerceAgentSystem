### 页面内跳转锚点

| 源页面 | 跳转目标 | 触发条件 |
|--------|---------|---------|
| Dashboard 状态条 | → /setup | 店铺数=0 或 Agent 数=0 |
| Dashboard 成果卡片 | → /agents/:agentType | 点击 topContributor agent |
| Dashboard 近期任务 | → /agents/:agentType | 点击任务行 |
| Dashboard 试用横幅 | → /settings/billing | 点击「升级套餐」 |
| Agent 列表 | → /agents/:agentType | 点击「配置」按钮 |
| Agent 列表 | → /agents/:agentType | 点击活跃任务数 Tag |
| Agent 配置页 | → /agents/exceptions | 异常链接 |
| 商品管理 Agent 提示 | → /agents | Agent 联动提示 |
| 异常中心 订单异常卡 | → /orders | 点击订单异常 |

## 7. 关联改动检查清单

当修改以下模块时，必须检查关联模块：

| 改动模块 | 必须检查的关联模块 |
|----------|-------------------|
| **router.tsx** | AppShell 菜单 selectedKeys、页面内 navigate() 跳转、旧路由重定向 |
| **AppShell.tsx** | router 路由定义、i18n nav.* keys、页面间跳转链接 |
| **domain.ts 类型** | 所有 API 文件、所有使用该类型的页面组件、MembersSettingsPage（角色） |
| **domain.ts Member.role** | MembersSettingsPage（roleOptions、角色展示）、i18n member.role.*、mockData members[]、demoMockData members[] |
| **product-design.md Agent 表格** | AgentListPage（Agent 数量）、i18n agent.* keys、SetupConfigPage（Agent 场景映射） |
| **dashboardApi** | DashboardPage、AppShell（轮询） |
| **agentsApi** | AgentListPage、AgentConfigPage、SetupConfigPage、DashboardPage |
| **storesApi** | StoreListPage、StoreDetailPage、SetupConfigPage、DashboardPage |
| **financeApi** | BillingSettingsPage、DashboardPage、AgentListPage |
| **i18n.tsx** | 所有使用对应 key 的组件（通过 Grep 全局搜索 key 名），包括 MembersSettingsPage 的 roleOptions |
| **mockData.ts** | 所有引用 `stores`/`tasks`/`approvals` 的 API + 页面 + members[] 角色值 |
| **agentMockData.ts** | agentsApi、AgentListPage、AgentConfigPage（含 15 个 Agent 配置） |
| **demoMockData.ts** | 所有使用 `useDemoMode()` 的组件 |
| **tokens.css** | global.css、providers.tsx（AntDesignThemeProvider） |
| **global.css** | 所有使用对应 CSS 类名的组件 |
