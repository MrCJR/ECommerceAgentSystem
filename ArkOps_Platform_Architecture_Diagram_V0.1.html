ArkOps Platform Architecture · V0.1

整体平台架构图

该图描述 ArkOps 在内测阶段的整体系统边界：SaaS 控制面负责租户、店铺、任务、审批、审计和额度；MuleRun 负责高价值 Agent 调试、浏览器沙盒和会话复用；未来可通过 Agent Runtime Adapter 平滑扩展到自研 Claw Runner。

1

用户入口层

租户管理员、运营人员和审批人从 Portal 进入系统。

ArkOps Portal React + TypeScript + Ant Design
登录与权限 Auth、Tenant Context、RBAC、成员管理
业务工作台 店铺授权、任务中心、审批中心、审计查询
外部通知通道：Webhook / Feishu / DingTalk / WeCom 审批提醒、登录失效、任务失败、异常告警
↓
2

ArkOps SaaS 控制面

后端使用 Node.js + NestJS + TypeScript，负责业务主控、任务编排和 runtime 解耦。

API Gateway / BFF REST API、Portal 聚合接口、回调入口
Auth & Tenant Context 登录态、租户上下文、权限守卫
Tenant / User / RBAC 租户、成员、角色、权限
Store Authorization 店铺、connectToken、session binding、re-auth
Task Control Plane Task、Workflow、Run、Step、状态机
Approval Service 高风险动作审批、超时取消、审批留痕
Billing & Quota 套餐、额度、Token、操作次数、Worker 限制
Audit & Artifact 审计日志、截图、Trace、报表、执行证据
Agent Runtime Adapter 把 ArkOps 标准任务转换为 MuleRun / Claw Runner 可执行格式，并把 runtime 事件转换回 ArkOps 时间线。
2A

数据与存储

支撑 SaaS 控制面运行的核心存储、队列、安全和可观测能力。

PostgreSQL + Prisma 租户、店铺、任务、审批、审计、计费
Redis + BullMQ connectToken TTL、队列、锁、异步任务
S3-compatible Storage 截图、浏览器 Trace、日志归档、报表文件
HashiCorp Vault 店铺凭证、Webhook 密钥、代理配置、LLM Key
Observability OpenTelemetry、Prometheus、Grafana、Loki
connectToken -> MuleRun session binding -> task payload -> agent events -> approval / audit / artifacts

3

Agent Runtime 与外部平台层

内测阶段先用 MuleRun；未来通过相同 adapter 接入自研 runtime，并面向多电商平台执行任务。

MuleRun Beta Runtime Web 手动启动 Agent、浏览器沙盒、profile/session 复用
Login Bootstrap Agent 人工登录、captcha / 2FA、session 建立与绑定
Advertising Agent ROI / CPC / CTR 监控，预算建议，素材替换
Product Launch Agent 标题 SEO、属性填写、详情页结构、A/B 测试
CRM / Finance Agent 用户召回、优惠券、利润核算、风险库存
Future Claw Runner 自研 Worker、Playwright、Docker/K8s 隔离、可回放执行
外部电商平台：TikTok Shop / Amazon / Shopify / 淘宝 / 抖音 / 京东 / Shopee / Lazada / Walmart Agent Runtime 通过浏览器沙盒、平台 API 或后续自研 Claw Runner 对接不同平台。

内测阶段关键闭环

1 添加店铺 用户在 ArkOps 创建 TikTok Shop 店铺授权记录。
2 生成 connectToken ArkOps 生成短期一次性绑定令牌。
3 MuleRun 登录沙盒 用户手动打开 Login Bootstrap Agent 并完成登录验证。
4 绑定 session MuleRun 回调 ArkOps，保存 runtimeSessionId 引用。
5 执行 Agent 任务 ArkOps 下发任务，MuleRun 执行并回传事件、截图和结果。
